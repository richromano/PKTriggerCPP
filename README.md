This is a conversion of:
https://github.com/asalamon74/pktriggercord

I converted it to a Windows VS build environment and ported it to C++.  I also made a few versions:
1. A command line version PKTriggerCordCLI
2. A DLL version PKTriggerCordDLL
3. A command line that uses the DLL PKTriggerCordConsole
4. A C# version - which I then used for my ASCOM PKTriggerCord driver: PKTriggerCordCSharp

JSON support was difficult to port and their may be errors.  I have not tested at all.

License remains from the old version.
